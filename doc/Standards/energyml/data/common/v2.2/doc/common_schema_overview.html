Common Data Schema Overview

Version 2.2

An Energistics Data Object Schema is a complete, high-level schema from one of the EnergyML families. In EnergyML, there is usually a single root element defined in each schema, though this is not a requirement of XML schemas. A Data Object is a single instance document for one of these schemas. This specification is primarily concerned with standards and guidelines for naming, or identifying these data objects, references within a data object to another data object, and references between a data object and other systems or naming authorities.

Common Schemas:

    

Abstract data object

  
The Abstract package contains the base XSD types from which all EnergyML Data Objects are derived.

This Package contains common re-usable complex types used throughout different Energistics standards.

CRS data object

  
This Package contains encapsulations of coordinate reference systems defined by GML, EPSG, well-known types as defined by ISO 19162, or other well-known text.

DataAssurance data object

  
A metadata object that can be bound to any reference object and carry data assessment information.
  XSD


EmlAllObjects data object

  
This Package includes all individual Common data objects in 1 file, ideal for importing all of them using a common schema location.
  XSD


GraphicalInformation data object

  
This Package contains the base schema of the graphical information object.
  XSD


LegacyUnitsOfMeasure data object

  
This Package contains customary legacy units.
  XSD


MeasureType data object

  
These types represent numeric quantities with a unit of measure implemented as a uom attribute. The units are gathered into single quantity class which represents a group of units. The unit symbols appropriate for the class are captured as enumerated lists. The implementation of this is to develop a type for each uom class that is a union of a standard list, the unknown value, and the Other: extension. Additional files will contain these structures.
  XSD


ObjectReference data object

  
This package contains the types and elements to allow an EnergyML Data Object to refer to another EnergyML Data Object (i.e. External References).
  XSD


QuantityClass data object

  
This file defines a set of standard POSC units of measure for various quantity classes. This units list captures information contained in the POSC units dictionary at http://www.posc.org/refs/poscUnits20.xml. The enumerated lists do not contain any deprecated units.
  XSD


ValueTypes data object

  
The collection of standard temperature and pressure values.
