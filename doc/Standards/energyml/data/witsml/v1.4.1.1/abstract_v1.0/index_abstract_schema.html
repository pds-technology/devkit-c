Energistics Abstract v1.0 Data Schema Files

Purpose

The abstract schema provides abstract types that can be utilized in multiple domain specific schemas. Currently, these types only support two element substitution groups. These substitution groups are designed to be used with the Energistics aggregate schema which is designed to support multiple domain messages within one generic wrapper message.

Details

Type abstractObject defines the abstract type which is the basis for the substitution groups. The type of all members of a substitution group must be derived from this abstract type.

Global element abstractContextualObject defines a substitution group for schemas that provide a business context for a collection of messages in an aggregate message. This is intended to define the what, when, why, who and where (etc) that is represented by the collection.

Global element abstractDataObject defines a substitution group for elements that may be part of the collection of messages in an aggregate message. For example, all WITSML plural global elements are a member of this substitution group.
